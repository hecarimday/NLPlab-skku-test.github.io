게시판 올리기 사이트: https://sweet-genie-b1cd1b.netlify.app/admin/

현재 기존의 흔적기관들을 발견해 그 코드를 이용해
홈페이지 위에 Boards, Links 추가

Papers도 존재하긴 했지만 어디에 쓸진 몰라서 지워놨음
Links는 일단 쓸모가 있을 수도 있다는 생각이 남겨놓음
Boards는 현재 학교 게시판처럼 보이게 하는데는 성공함

다만 게시물을 관리에서 고려해야할점
1. 보안문제
글을 작성할려면 로그인같이 신원 확인이 필요한데 이게 쉽지 않음

2. 글쓰는 방식
네이버 카페처럼 사이트 내에서 글을 쉽게 쓰도록 하기 힘듬

따라서 글쓰기 방법은 2가지로 예상
1. github에 _posts에서 자신이 올릴 글을 markdown으로 적기
2. netlify에 연동을 해서 별도에 사이트에서 권한을 일부 사람들에게 부여하고, 거기서 작성을 하기

1번 방식은 markdown으로 적어야 해서 적기 힘들지만 깃허브 권한만 있으면 랩실 누구나 변경 가능
2번 방식은 네이버 카페 글쓰기 처럼 쉽게 적을 수 있지만 netlify에 따로 연동하기 힘듬 + 접근자 제한이 있음

해야할것
1. 공지 검색(키워드 제목 등으로)
2. 게시물 페이지 제한을 둬서 넘치는 것은 다음 페이지로 넘기기
3. 사진 파일, 문서 파일, 영상 파일, 유투브 링크 올려지는지 확인하기
